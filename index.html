Hello, Moon! in 10 lines

This is the entire framework. Pages hold objects, objects hold layers. That's it.

javascript 
const rocket = require('crescent-js');

// Create a page
const page = rocket.page({ page_id: 'home', title: 'My App' });

// Add content
const heading = rocket.object({ object_id: 'heading' });
heading.add_layer(
  rocket.layer({
    layer_type: 'text',
    layer_id: 'title',
    text: 'Hello, Moon!',
    tag: 'h1'
  })
);
page.add(heading);

// Render
console.log(page.render());

Run your app

One command. Pick a port, or let it default to 3000.

bash 
crescent run <FILE_PATH> 3030   # Run on a specific port
crescent run <FILE_PATH>        # Auto-runs on port 3000
