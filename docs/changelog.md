# Changelog

## [2.0.0] - 2025-01-01

### Added
- Full-stack framework with frontend, backend, and auth modules
- Page, Object, and Layer system for building UIs
- Base, Text, Image, Shape, and Input layer types
- Transition animations (fade, slide, scale, rotate)
- Event triggers (click, hover, focus, submit, change, keydown)
- Responsive design breakpoints
- Backend functions, conditionals, and loops
- API call and API make for external and internal APIs
- Collect for data aggregation
- Boolean logical operations
- Authentication system with signup and login
- OAuth support for third-party providers
- Password hashing and verification
- Cookie-based session management
- Global utility functions (math, print, redirect, etc.)
- Renderer for HTML output

---

## [1.0.0] - 2024-06-01

### Added
- Initial release
- Basic page and object system
- Text and base layers
- Simple function and conditional support
